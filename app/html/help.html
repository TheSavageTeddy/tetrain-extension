Default Controls

Arrow Left • Move Piece Left

Arrow Right • Move Piece Right

Arrow Up • Rotate Right

Arrow Down • Soft Drop

Space • Hard Drop

C • Hold

P • Pause Game

Q • Quit Game

Enter • Start/Restart

Esc • Exit popup


FAQ

Q: Why did my game crash/freeze?

A: This is a bug, which should/will be fixed soon. You can press 'q' to go back and play to start again.

Q: How do I disable hold or next pieces?

A: Go into settings to disable or enable them

Q: Can I change the window size?

A: Yes you can, we have 3 window sizes. Go into settings to change them.

Q: Can I change the controls used?

A: Yes, in settings, scroll down for custom controls.

Scoring

1 line = 100 * level

2 lines = 300 * level

3 lines = 500 * level

4 lines = 800 * level

2 or more 4 lines in a row = 1200 * level

Soft drop = 1 point per block dropped (Does not get multiplied by level)

Hard drop = 2 points per block dropped (Does not get multiplied by level)
